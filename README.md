# 🎲 YAZY

A browser-based Yahtzee-inspired game built using HTML, CSS and JavaScript.

YAZY supports solo play, team-based gameplay, score tracking, custom player names, and a unique Lucky Number mechanic that adds an extra layer of unpredictability to every game.

---

## Features

* Dice rolling and locking system
* Solo and Team game modes
* Automatic score calculation
* Turn and round management
* Winner detection and winner screen
* Lucky Number bonus system
* LocalStorage-based state management
* Responsive UI built with HTML, CSS and JavaScript


## Tech Stack (so far)

* HTML5
* CSS3
* JavaScript (Vanilla JS)

## Game Rules

YAZY follows most traditional Yahtzee scoring mechanics:

* Ones to Sixes
* Three of a Kind
* Four of a Kind
* Full House
* Straight
* YAZY (Five of a Kind)

In addition, YAZY includes a **Lucky Number** feature that can award bonus points during gameplay.
